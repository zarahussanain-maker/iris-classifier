# Iris Classifier (Decision Tree)

## Overview
End-to-end ML example that builds a decision-tree classifier on the classic 
Iris dataset using scikit-learn.
## Quick Start

```bash
git clone https://github.com/zarahussanain-maker/iris-classifier.git
cd iris-classifier
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python src/train.py
```
